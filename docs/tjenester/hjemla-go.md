# Hjemla GO

Hjemla GO er en AR-basert mobilapp: pek telefonen mot boliger du går forbi og se verdiestimatet. Appen utvider plattformen fra research ved skjermen til utforsking ute i felten.

## Funksjoner

- Verdiestimat-oppslag i sanntid mens du er på farten.
- Samme estimat og boliginformasjon som på web.
- Tilgjengelig i App Store.

:::tip
Appen bruker de samme dataene som hjemla.no, så det du ser ute samsvarer med boligsiden på nett.
:::

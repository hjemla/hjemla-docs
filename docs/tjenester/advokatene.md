# Hjemla Advokatene

Er det noe med boligen din som ikke stemmer? Hjemla-advokatene er spesialister på boligrett og reklamasjonssaker, og kan bistå bredt innenfor fast eiendom — i samarbeid med Advokatfirmaet Solver.

## Hva de hjelper med

- Mangler oppdaget etter boligkjøp (reklamasjon).
- Uavhengig «second opinion» ved avslag fra forsikringsselskapet.
- Bred bistand innen fast eiendom ved kjøp og salg.

## Slik tar du kontakt

- Send inn saken din kostnadsfritt via `advokat.hjemla.no/registrer`.
- Ring 22 44 10 00 (hverdager 09–15).

:::tip Fritt advokatvalg
Har du boligkjøper- eller innboforsikring, bestemmer du selv hvem som tar saken din.
:::

# Boligkart

Boligkartet er hovedflaten i Hjemla. Her ser du boliger til salgs samtidig som du ser hva som nylig ble solgt i samme område — noe som gir unik innsikt i hva som faktisk har skjedd i markedet.

## Statuser i kartet

| Status | Beskrivelse |
| --- | --- |
| Til salgs | Boliger som er annonsert for salg akkurat nå. |
| Solgt | Nylig solgte boliger, med salgspris der den er tilgjengelig. |
| Vurderes solgt | Boliger eier har markert som mulige for salg. Se [Vurderes solgt](./vurderes-solgt.md). |

## Filtre og navigering

- Filtrér på status (til salgs / solgt) og prisintervall.
- Zoom og panorer for å utforske byer, bydeler og nabolag.
- Klikk på en bolig for å åpne [eiendomssiden](./eiendomssiden.md) med detaljer og estimat.

:::tip
Kart-URL-en lagrer posisjon, zoom, status og pris. Det betyr at du kan dele en eksakt kartvisning ved å kopiere lenken.
:::

## Slik bruker du det

1. Søk på adresse eller lim inn en Finn-lenke i søkefeltet.
2. Velg ønsket status og prisintervall.
3. Sammenlign solgte og annonserte boliger for å vurdere prisnivå.

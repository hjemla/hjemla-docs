# Eiendomssiden

Når du klikker på en bolig i kartet, åpnes eiendomssiden. Her samles all tilgjengelig informasjon om boligen ett sted, med data fra offentlige registre, annonser, salg og eier.

## Hva siden viser

| Seksjon | Innhold |
| --- | --- |
| Fasiliteter | Balkong, parkering, heis, hage og andre fasiliteter. |
| Egenskaper | Boligtype, byggeår, eierform og standard. |
| Størrelse | Bruksareal (BRA), antall rom, soverom og bad. |
| Kostnader | Felleskostnader, gebyrer og andre løpende utgifter. |
| Salgshistorikk | Tidligere salg og prisutvikling for adressen. |
| Verdiestimat | [Hjemlaestimatet](./hjemlaestimatet.md) med komponentene Historisk markedsverdi, Egenskaper og Beliggenhet. |

:::tip
Er du eier? Overta boligen i [Min bolig](./min-bolig.md) for å rette opplysninger og følge verdien over tid.
:::

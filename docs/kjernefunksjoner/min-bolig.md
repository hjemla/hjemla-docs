# Min bolig

Min bolig er din egen, innloggede arbeidsflate. Overta adressen din og lås opp et privat dashbord for boligen.

## Hva du kan gjøre

- **Følg verdien** — se boligens estimerte markedsverdi og hvordan den endrer seg over tid.
- **Rediger boligdata** — rett fasiliteter, egenskaper og dokumentasjon. Estimatet beregnes på nytt.
- **Følg nabolaget** — se naboenes salgspriser og hva som skjer i området ditt.
- **Vurderes solgt** — marker boligen som mulig for salg for å teste interessen.

## Kom i gang

1. Logg inn på Hjemla.
2. Søk opp adressen din og velg «Min bolig».
3. Bekreft at du eier boligen, og fullfør overtakelsen.
4. Oppdater opplysninger for et mer presist estimat.

:::note
Denne funksjonen krever en konto. Se [Konto og innstillinger](../konto-og-hjelp/konto-innstillinger.md).
:::

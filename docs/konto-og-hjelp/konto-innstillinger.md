# Konto og innstillinger

Det meste av Hjemla er åpent og gratis. Noen funksjoner krever at du logger inn.

## Tilgangsnivåer

| Funksjon | Tilgang |
| --- | --- |
| Boligkart (til salgs / solgt) | Åpent, uten konto |
| Hjemlaestimatet for alle boliger | Åpent, uten konto |
| Nabolag og boligpriser | Åpent, uten konto |
| Min bolig | Krever innlogging |
| Vurderes solgt | Krever innlogging (gratis) |

## Personvern og vilkår

Personvern, brukervilkår og vilkår for kjøp finner du i bunnen av hjemla.no.

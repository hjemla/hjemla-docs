# Data og kilder

Hjemla samler og kombinerer informasjon fra flere kilder. Grunnlaget er offentlige eiendomsdata, beriket med annonse-, salgs- og eieropplysninger.

## Kilder

| Kilde | Hva den bidrar med |
| --- | --- |
| Ambita | Offentlige eiendomsdata (matrikkel m.m.) — grunnlaget for boliginformasjonen. |
| Annonseplattformer og meglerhus | Sanntidsinformasjon om boliger til salgs, prisantydning og beskrivelser. |
| Offentlige registre | Historiske salg og offisielle opplysninger. |
| Boligeiere | Egenrapporterte opplysninger om egen bolig via «Min bolig». |

## Hvordan boliginformasjon er organisert

For hver bolig deles informasjonen inn i fasiliteter, egenskaper, størrelse og kostnader. Når eier oppdaterer opplysninger, beregnes verdiestimatet på nytt.

:::caution Merk
Hjemla er en del av Placepoint AS. Estimater og opplysninger er veiledende og bør suppleres med profesjonell vurdering ved viktige beslutninger.
:::

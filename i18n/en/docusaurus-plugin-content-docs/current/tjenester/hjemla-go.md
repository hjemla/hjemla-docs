# Hjemla GO app

Hjemla GO is an AR-based mobile app: point your phone at homes you walk past and see the value estimate. The app extends the platform from desk research to exploring out in the field.

## Features

- Real-time value lookup while you're on the move.
- Same estimate and property information as on the web.
- Available on the App Store.

:::tip
The app uses the same data as hjemla.no, so what you see outside matches the property page online.
:::

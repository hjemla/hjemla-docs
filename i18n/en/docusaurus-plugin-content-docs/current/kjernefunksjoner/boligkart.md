# Property map (Boligkart)

The property map is the main surface in Hjemla. You see homes for sale at the same time as what recently sold in the same area — giving unique insight into what has actually happened in the market.

## Statuses on the map

| Status | Description |
| --- | --- |
| For sale | Homes currently advertised for sale. |
| Sold | Recently sold homes, with sale price where available. |
| Considered for sale | Homes an owner has flagged as possibly for sale. See [Considered for sale](./vurderes-solgt.md). |

## Filters and navigation

- Filter by status (for sale / sold) and price range.
- Zoom and pan to explore cities, districts and neighbourhoods.
- Click a home to open the [property page](./eiendomssiden.md) with details and the estimate.

:::tip
The map URL stores position, zoom, status and price — so you can share an exact map view by copying the link.
:::

## How to use it

1. Search an address or paste a Finn link in the search box.
2. Choose the status and price range you want.
3. Compare sold and listed homes to judge the price level.

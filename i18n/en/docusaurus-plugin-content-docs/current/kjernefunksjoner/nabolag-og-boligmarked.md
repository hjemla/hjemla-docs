# Neighbourhoods & market

Area pages turn raw transactions into trends — for cities and districts across the country.

## Page types

| Page | Description |
| --- | --- |
| Neighbourhood | Area pages for cities and districts with a profile and key figures. |
| Home prices | Average prices and price development by area. |
| Property transfers | Registered changes of ownership / transactions. |
| Housing market | Activity and market overview to judge if a home is fairly priced. |

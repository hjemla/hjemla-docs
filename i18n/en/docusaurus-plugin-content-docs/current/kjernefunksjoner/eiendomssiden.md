# The property page

When you click a home on the map, the property page opens. All available information about the home is gathered in one place, with data from public registries, listings, sales and the owner.

## What the page shows

| Section | Content |
| --- | --- |
| Facilities | Balcony, parking, elevator, garden and other amenities. |
| Attributes | Property type, build year, ownership form and standard. |
| Size | Usable area (BRA), number of rooms, bedrooms and bathrooms. |
| Costs | Shared costs, fees and other running expenses. |
| Sales history | Past sales and price development for the address. |
| Value estimate | The [Hjemla estimate](./hjemlaestimatet.md) with the Historical market value, Attributes and Location components. |

:::tip
Are you the owner? Claim the home in [My home](./min-bolig.md) to correct information and track its value over time.
:::

# Account & settings

Most of Hjemla is open and free. Some features require you to log in.

## Access levels

| Feature | Access |
| --- | --- |
| Property map (for sale / sold) | Open, no account |
| Hjemla estimate for all homes | Open, no account |
| Neighbourhoods and prices | Open, no account |
| My home (Min bolig) | Requires login |
| Considered for sale | Requires login (free) |
| Hjemla GO | Free app |

## Privacy & terms

Privacy policy, terms of use and purchase terms are found at the bottom of hjemla.no.

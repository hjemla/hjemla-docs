# Data & sources

Hjemla collects and combines information from several sources. The foundation is public property data, enriched with listing, sales and owner information.

## Sources

| Source | What it contributes |
| --- | --- |
| Ambita | Public property data (matrikkel etc.) — the basis of the property information. |
| Listing platforms & brokerages | Real-time data on homes for sale, asking prices and descriptions. |
| Public registries | Historical sales and official records. |
| Home owners | Self-reported details about their own home via "My home". |

## How property information is organised

For each home the information is grouped into facilities, attributes, size and costs. When an owner updates information, the value estimate is recalculated.

:::caution Note
Hjemla is part of Placepoint AS. Estimates and information are indicative and should be supplemented with professional assessment for important decisions.
:::
